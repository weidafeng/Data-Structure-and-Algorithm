# 二分查找
复杂度： logn

## 适用情况：
1. 有序数组/字符串
2. 存在上下界（因为要通过上下界求中间的元素）
3. 元素可以通过索引访问

## 模板（伪代码）

```python

left, right = 0, len(array) - 1
while left <= right:
    mid = (right - left) // 2 + left # 求中间元素（也可以写为 （left+rigjht)//2
```
